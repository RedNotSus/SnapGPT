ChatGPT Screenshot Uploader Test Page

Hotkey Test

Press Ctrl+Shift+Y (Windows) or Command+Shift+Y (Mac) on this page to test the screenshot functionality.

If it works, ChatGPT should open with this page as a screenshot.

Troubleshooting

If the hotkey doesn't work:

  1. Open Chrome's extension page: chrome://extensions
  2. Find this extension and click on "Details"
  3. Check "Allow in Incognito" if you're trying to use it there
  4. Make sure "Site access" is set appropriately
  5. Click "Extension options" and verify shortcuts
  6. Try clicking the extension icon in the toolbar (should also work)

Console Output

Check Chrome's DevTools console for the extension by:

  1. Go to chrome://extensions
  2. Enable "Developer mode" (top right)
  3. Click on "background page" link under this extension
  4. Look for debug messages in the console
