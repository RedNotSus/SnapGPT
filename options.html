SnapGPT

Keyboard Shortcut

Chrome requires changing extension shortcuts at chrome://extensions/shortcuts.

Open the shortcuts page: Open shortcuts

Look for “Capture current tab & upload to ChatGPT” and set your preferred keybinding.

Behavior

  • Captures the visible area of the active tab (Chrome API limitation).
  • Reuses an existing ChatGPT tab if present; otherwise opens a new one.
  • Uploads the screenshot and attempts to auto-send.

Troubleshooting

If the hotkey doesn't work:

  1. Make sure the extension is enabled on chrome://extensions
  2. Verify no other extension uses the same keyboard shortcut
  3. Try using the extension icon in the toolbar as an alternative
  4. Check the browser console for errors (right-click > Inspect > Console)
  5. For advanced troubleshooting, view the service worker logs:
    • Go to chrome://extensions
    • Enable "Developer mode" (top right)
    • Click "service worker" link under this extension
    • Look for debug messages or errors in the console
  6. Try our test page to see if the extension works there
